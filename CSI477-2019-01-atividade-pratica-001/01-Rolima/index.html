Apuração dos dados de uma corrida de carrinhos de Rolimã!

O objetivo desse sistema é realizar a apuração de uma competição de carrinhos de rolimã, você poderá inserir os dados de todos os participantes bem como o tempo que cada competidor obteve na competição, com isso, o sistema irá informar quais foram os vencedores...

Dessa forma o objetivo do sistema é apenas realizar a apuração da competição, e se trata apenas de uma forma bem atraente de divulgação da mesma para os competidores se os organizadores acharem conveniente !
